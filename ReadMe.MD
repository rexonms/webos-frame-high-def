# Getting Started

## Open the Simulator
Go to Directory 

    cd /opt/webOS_TV_SDK/Emulator/v5.0.0
    open .
    run `LG_webOS_TV_Emulator_RCU`

This could also be in VirtualBox

## Run the code in Simulator
Open the terminal

    rnv run -p webos
The app can also be accessed via http://0.0.0.0:8088/
